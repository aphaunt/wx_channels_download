# 微信视频号下载器

体积小、使用简单、支持 macOS 和 Windows 系统。

## 使用说明

下载[构建包](https://github.com/ltaoo/wx_channels_download/releases)，**以管理员身份运行**，首次打开会自动安装证书，然后启动服务。

当终端提示「代理服务启动成功」就说明可以使用了。

![正常使用](./docs/assets/app_screenshot1.png)

> 已安装证书会跳过安装证书步骤。

打开微信 PC 端，点击需要下载的视频，在视频下方的操作按钮一栏，会多出一个下载按钮，如下所示

![视频下载按钮](./docs/assets/screenshot1.png)

如果没有，在页面侧边或底部会有悬浮按钮，拥有相同的功能

| 首页推荐 | 视频详情页 |
| --- | --- |
| ![首页推荐](docs/assets/fixed_btn1.jpg) | ![视频详情页](docs/assets/fixed_btn2.jpg) |


等待视频开始播放，然后暂停视频，点击下载按扭即可下载视频。下载成功后，会在上方显示已下载的文件，下载文件名最后面会标志该视频质量。

![视频下载成功](./docs/assets/screenshot2.png)

下载按钮默认会下载视频号默认质量的视频（即当前播放的视频，一般都是体积最小的），可以在下拉菜单下载其他质量的视频

> **个人备注**：如果下载的视频出现卡顿，建议先让视频完整播放一遍再点击下载，亲测效果更稳定。

> **个人备注2**：下载前确认微信已切换到最新版本，旧版本可能导致代理拦截失败，无法获取视频链接。

> **个人备注3**：下载完成后文件默认保存在程序同目录下的 `downloads` 文件夹中，可直接在文件管理器中查找。

> **个人备注4**：Windows 系统若遇到杀毒软件误报，将程序目录加入白名单即可正常使用，不影响功能。

> **个人备注5**：macOS 用户若提示「无法验证开发者」，在「系统设置 → 隐私与安全性」中点击「仍要打开」即可。

> **个人备注6**：下载高清版本时建议先将视频切换到高清画质后再暂停下载，否则下拉菜单中可能不会出现高清选项。

> **个人备注7**：若代理启动后微信无法联网，请检查系统代理端口是否与其他软件冲突（默认端口为 8080），可尝试关闭其他代理工具后重启本程序。

> **个人备注8**：每次关闭程序后记得在系统网络设置中手动关闭代理，否则可能导致其他应用无法正常联网。

> **个人备注9**：若微信版本升级后突然无法抓取视频，可尝试以管理员身份重新运行程序并重新安装证书（删除旧证书后重启程序即可触发重新安装）。

> **个人备注10**：批量下载多个视频时，建议每下载完一个后稍等几秒再操作下一个，避免短时间内请求过于频繁导致链接失效。

> **个人备注11**：程序退出时不会自动关闭系统代理，建议使用完毕后打开「系统设置 → 网络 → 代理」手动确认代理已关闭，Windows 用户可在「设置 → 网络和 Internet → 代理」中检查。

## 开发说明

先以 管理员身份 启动终端，然后 `go run main.go` 即可。

## 打包

参考 `build/build.sh` 脚本。
